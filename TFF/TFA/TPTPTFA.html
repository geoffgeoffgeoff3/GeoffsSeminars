TFA Problems in the TPTP

Synopsis

  • First release: TPTP v5.0.0 - September 2010
  • TPTP v6.1.0 - 1757 TFF problems, including 1046 TFA

Domains

  • ARI: Arithmetic for testing ATP systems
  • DAT: Data structures
    • Lists of integers
    • Arrays of integers with integer indices
  • GEG: Geography
    • Distances between cities
  • HWV: Hardware verification
    • Intel robot verification
    • SPARCT2 RTL hardware design verification
  • MSC: Miscellaneous
    • Celcius to Fahrenheit conversion
  • NUM: Number theory
    • Constraint modeling
    • Properties of arithmetic operators
    • Collatz conjecture
  • PUZ: Puzzles
    • N queens
  • SET: Set theory
    • Sizes of various sets
  • SWV: Software verification
    • Finite state model checking safety properties
    • Verification of algorithms, e.g., Binary search, Bresenham, Dijkstra, Fibonacci, GCD, Insertion sort, Merge sort, ...
  • SYN: Syntactic
    • Facts about integers
    • Pigeon hole problem
