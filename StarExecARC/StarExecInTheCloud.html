StarExec in the Cloud

In Kubernetes

  • Containerised ATP systems in a public repository
  • Abstraction layer eliminating MariaDB-specific bindings
  • Head node → Kubernetes service
  • Compute nodes → Kubernetes jobs
  • Kubernetes job scheduling

On AWS

  • Kubernetes control plane → AWS Elastic Kubernetes Service
  • Head and compute nodes → Amazon EC2 instances
  • Database → Amazon Relational Database (RDS)
  • File system → Amazon Elastic File System
  • ATP systems containerisation → Amazon Trusted Solver format
