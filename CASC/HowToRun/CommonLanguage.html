A Common Language

The Need for a Common Language

  • Used for writing problems, and solutions
  • Tools to initially export to other languages
  • Systems should be able to easily add parsers
  • Provides for equal presentation of problems to systems

The TPTP Language

  • Same language for input (problems) and output (derivations, models)
  • Prolog like with a simple concrete syntax
  • Extended BNF is lex/yacc/flex/bison friendly
  • Hyperlinked BNF at tptp.org/TPTP/SyntaxBNF.html
    (Local copy)
  • TPTP2X and TPTP4X export from TPTP to other languages
