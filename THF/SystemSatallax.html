Satallax

Overview

  • Generates propositional clauses for a tableau calculus
  • Cooperates with the SAT solver MiniSat
  • Implemented in Steel Bank Common Lisp

TPTP-THF Features

  • First THF system built after TPTP-THF developed
  • THF0 is the native language for input
  • Debugged using THF problems in the TPTP
