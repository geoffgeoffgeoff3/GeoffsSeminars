Typed Higher-order Form

Status Quo

  • Many frameworks, Church's simple type theory is a common basis
  • Applications in verification, KB reasoning, mathematics, logics
  • Proof assistants: Isabelle, HOL, Coq, ProofPower, Omega, ...

The THF Language

  • Syntactically conservative extension of FOF language (Prolog-like)
  • Layers - THF0, THF, THFX
  • Example 
    thf(const_in,type,( 
    in: ( $i > ( $i > $o ) > $o ) )).

thf(ax_in,axiom,( 
    ( in = ( ^ [X: $i,S: ( $i > $o )] : ( S @ X ) ) ) )).
