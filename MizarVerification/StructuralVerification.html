Structural Verification with GDV

Assumptions

  • Assumption (supposition in Jaskowski's terminology): unproved proposition introduced (assumed to be true) locally in proof
  • Coupled with a discharge (elimination) rule, making the reasoning result unconditionally true again
  • Frequent example is proof by contradiction: asssume negation, deduce contradiction, i.e.: not C |- $false, then discharge not C: |- not C => $false, then |- C

TPTP Syntax

  • formula role assumption
  • asssumptions([...]) list for recording asssumptions
  • discharge_asm list of discharged assumptions

Example

fof(e1,assumption, subset(c1,c2), introduced(assumption)). 

fof(i3,plain, c2 = set_union2(c1,set_diff(c2,c1)), 
  inference(conclusion,[status(thm),assumptions([dt_c1,e1])],[e3,i4])). 

fof(i2,plain, (subset(c1,c2) => c2 = set_union2(c1,set_diff(c2,c1)), 
  inference(discharge_asm,[status(thm),assumptions([dt_c1]), 
    discharge_asm(discharge,[e1])], [e1,i3])).
