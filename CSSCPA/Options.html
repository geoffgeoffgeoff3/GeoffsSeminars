Three Design Options

In-house components

  • Easy to manage
  • Unified data format
  • Flexible communication options
  • Require expertise and programmer power

Modified existing components

  • Can be modified for management
  • Can insert data format translation
  • Can insert communication hooks
  • Can use existing high performance systems
  • Require porting, porting, porting

Unmodified existing components

  • Can be difficult to manage
  • Need external data format translation
  • No facilities for communication
  • Can use existing high performance systems
  • Can be used directly, without sources, and are easily updated
