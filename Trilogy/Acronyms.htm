TPTP-CASC-SSCPA

TPTP

  • The TPTP Problem Library
  • A comprehensive library of the ATP test problems
  • Aims to move ATP system evaluation onto a firm footing
  • Necessary because some published results are misleading

CASC

  • The CADE ATP System Competition
  • Annual evaluation of ATP systems
  • Aims to evaluate and stimulate system development
  • Provides exposure within and outside the ATP community.

SSCPA

  • The Smart Selective Competition Parallelism ATP system
  • Uncooperative multiple calculus competition parallelism ATP
  • Aims to harness individual system's strengths
  • Relies on meaningful ATP system evaluation
