Using the TPTP World (--)

  • Writing TPTP Problems
  • SystemOnTPTP demonstration using PUZ001+1
    • Start at the TPTP : tptp.org
    • Browse TPTP problems to PUZ001+1
    • Note language, SPC, Rating
    • Browse TSTP solutions to E solution, note language
    • Could rush forward to "Solve problem", but first ...
    • Copy formulae, open SystemB4TPTP : http://tptp.org/cgi-bin/SystemB4TPTP, and explain interface. Then check syntax using default, and with BNFParserDrill.
    • Open SystemOnTPTP : http://tptp.org/cgi-bin/SystemOnTPTP, and explain interface. Then check axioms using Paradox.
    • Add conjecture and prove with E. Note yacko proof, then redo with "TPTP format". Note language.
    • Could shortcut using SoTSTP, but first ... Copy formulae, open SystemOnTSTP : http://tptp.org/cgi-bin/SystemOnTSTP, and explain interface.
    • Run IDV on solution. Explain IDV interface. Show interestingness. Show "good/bad cat lemmas". Show verification.
    • Then check proof using GDV.
