The Truth Evaluation Approach

  • A recursive translation function trs
    • For an RM3 formula F and target truth value V (one of T3, B3, F3) ...
          trs(F,V) caputures the conditions for F to be V

  • Example translation rules
    F trs(F,T3)     trs(F,B3)     trs(F,F3)
    φ φ+ ∧ ¬φ-     φ+ ∧ φ-     ¬φ+ ∧ φ-
    ¬φ trs(φ,F3)     trs(φ,B3)     trs(φ,T3)
    φ→ψ trs(φ,F3) ∨ trs(ψ,T3)     trs(φ,B3) ∧ trs(ψ,B3)     (complex)
    ∀X φ ∀X trs(φ,T3)     ∃X trs(φ,B3 ∧ ¬∃X trs(φ,F3))     ∃X trs(φ,F3)

  • Additional axioms
    • Exhaustion axiom for each predicate ∀ (φ+ ∨ φ-)
    • Definition axioms for each predicate ∀ (φ++ ↔ (φ+ ∧ ¬φ-)); ∀ (φ+- ↔ (φ+ ∧ φ-)); ∀ (φ-- ↔ (¬φ+ ∧ φ-))

  • Define des(F) ≅ trs(F,T3) ∨ trs(F,B3).
    Axioms ⊨RM3 Conjecture
        iff
    (des(Axioms) ∪ Exhaustion ∪ Definition) ⊨FOL trs(Conjecture)
  • Proof based on isomorphism between RM3 and FOL interpretations
