Tools for Interpretations

Viewing

  • To understand interpretations' structures and relationships
  • Interactive Model Viewer - IMV (like IDV)
  • One idea

Verifying

  • Geoff's Model Verifier - GMV (like GDV)
  • Check each formula is interpreted as true
  • Alternative way: check input + model is not UNS

Examining

  • To determine relationships, suggesting new models
  • Check for syntactic variants
  • Compare the sets of ground atoms that are true

Manipulating

  • Extract submodel for sublanguage
  • Extend model for added axioms
  • Combine models for combined axiomatizations
