The Problem Status Ontology

Problem Format

  • F of the form Ax => C
  • F is valid means Ax |= C
  • Otherwise there are many possibilities
  • A set of axioms (without a conjecture) is treated as $true => Ax
    • This is the case for a CNF set

Overview

  • Upper ontology
  • Deductive status
  • Preserving status
  • Unsolved status
