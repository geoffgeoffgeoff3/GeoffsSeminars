SigmaKEE
The Sigma Knowledge Engineering Environment

Overview

  • KBR environment for developing and using logical theories
  • Built to support SUMO
  • Runs as an Apache Tomcat service
  • Browser interface for users
  • Writen in Java and JSP

Using SigmaKEE

  • Uploaded knowledge bases indexed and preprocessed
  • Browsing - flat and graph browser
  • Querying - use ATP to find proofs and answers
  • Result presentation hyperlinked into knowledge base

Existing ATP in SigmaKEE

  • Customized Vampire
  • Has required features
    • Multiple answer extraction
    • Proof output as explanation
    • Multiple queries without reloading KB
    • Basic arithmetic
  • 5 years old and unsupported - SigmaKEE could use more
