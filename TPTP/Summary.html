The TPTP Problem Library

A Library of Test Problems

  • Aim
    • A simple, unambiguous, source and reference mechanism
    • Centralize problem collection and maintenance
    • Improve ATP system testing and evaluation

  • Motivation
    • Statistically significant results
    • Repeatability of testing
    • Meaningful system evaluations and comparisons

Features

  • First released Fri Nov 12 15:30:54 +1000 1993
  • TPTP 7.5.0 - 54 domains, 24098 problems
  • Three layers of logic FOF (+CNF), TFF (TF0+TF1), THF (TH0+TH1)
  • Documented: References, Status, Rating, Statistics,
  • Human and computer readable: Name, Role, Logic, Annotations
  • A key - the TPTP language
