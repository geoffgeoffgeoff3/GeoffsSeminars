Prototype Algorithms

  • Principles
    • In the context of a saturation prover
    • Minimize disruption to normal operation

  • Synchronous algorithm
    Store external data references in separate area
In the saturation loop
    If a negative literal is selected
        Foreach matching external reference
            Query external database
            Wait for external data to be delivered
            Add returned data to active list
    Continue normal processing

  • Asynchronous algorithm
    Store external data references in separate area
In the saturation loop
    Add any delivered external data in passive list
    If a negative literal is selected
        Foreach matching external reference
            Query external database
    Continue normal processing
