The Truth Evaluation Approach

  • A recursive translation function trs
    • For a logic XYZ, formula F and target truth value TV (of XYZ) ...
          trs(F,TV) captures the conditions for F to be TV

  • Example translation rules
    • Rules for atoms are fixed
    • Rules for unary and binary connectives generated from truth tables
    • Rules for quantified formulae based on the Truth Diamond

    F trs(F,T)   trs(F,B)   trs(F,N)   trs(F,F)
    φ φcT ∧ ¬φcF   φcT ∧ φcF   ¬φcT ∧ ¬φcF   ¬φcT ∧ φcF
    ¬φ trs(φ,F)    trs(φ,B)   trs(φ,N)   trs(φ,T)

     trs(φ,B)) ∧

  • Additional axioms
    • Definition axioms (for truth values of XYZ)
      ∀ (φT ↔ (φcT ∧ ¬φcF)); ∀ (φB ↔ (φcT ∧ φcF));
      ∀ (φN ↔ (¬φcT ∧ ¬φcF)); ∀ (φF ↔ (¬φcT ∧ φcF))
    • Exhaustion axiom (for truth values of XYZ)
      ∀ (φT ∨ φB ∨ φN ∨ φF)

  • Define des(F) ≅ Disjunction of trs(F,DTV) for Designated Truth Values of XYZ
    Axioms ⊨XYZ Conjecture
        iff
    (des(Axioms) ∪ Exhaustion ∪ Definition) ⊨FOL des(Conjecture)
    • Proof based on isomorphism between XYZ and FOL interpretations
