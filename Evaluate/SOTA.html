The State of the Art

A SOTA System

  • A state-of-the-art (SOTA) system has, for every problem, the performance of the best individual system
  • A SOTA system solves every problem solved by some system

SOTA Contributors

  • A system is a SOTA contributor if it is not subsumed
  • SOTA contributors run in parallel form a SOTA system
  • Number of SOTA contributions provides some overall insight
