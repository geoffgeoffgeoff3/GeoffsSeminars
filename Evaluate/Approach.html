Approach and Principles

Approach

  • Analysis not possible in the 1st order case
  • Empirical measurement based on ATP system performance is required
  • Problem rating is by and for ATP systems
    • Problem rating within SPCs
  • Only know ranking of systems within SPCs
    • Problem ratings based on only solutions found
  • Avoid influence by large number of weak systems
    • Use only SOTA contributors

Principles

  • If all systems solve a problem, it is as easy as possible; lowest rating
  • If no systems solve a problem, it is as hard as possible; highest rating
  • As more systems are able to solve the problem, the rating decreases
