Specialist Problem Classes

System Specialization and Generalization

  • Problems have easily identifiable characteristics
  • ATP systems are specialized to problem types
  • ATP systems can't attempt some problem types
  • ATP systems are suited to some problem types

Specialist Problem Classes (SPCs)

  • Problems that are homogeneous wrt ATP systems, e.g., TF0_THM_NEQ_ARI
  • Use an ATP system appropriate for problem's SPC
  • Analytical and empirical evidence that the SPCs are appropriate
